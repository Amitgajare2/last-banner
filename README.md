## Installing Commands 

1. `apt update && apt install git -y`
2. `git clone https://github.com/remo7777/T-Header.git`
3. `cd Last-Banner`
4. `ls`
5. `chmod +x *`
6. `./install`
7. `cd`
8. `cd ..`
9. `cd usr`
10. `cd etc`
11. `rm bash.bashrc`
12. `cd`
13. `cd Last-Banner`
14. `mv bash.bashrc /data/data/com.termux/files/user/etc`
15. `bash`
